# Lucide Common 100 Icon Pack

This package contains 100 commonly used Lucide SVG icons for agent avatar selection.

## Contents

- `icons/`: 100 SVG files
- `icon-names.json`: metadata + icon name list
- `icon-names.ts`: TypeScript constant + type
- `icon-names.rs`: Rust string slice constant
- `LICENSE`: Lucide ISC license

## Source

- Repository: https://github.com/lucide-icons/lucide
- Commit: 9a8e02375483a821e79cd77e7fe854f73c017604

## Notes

- Icon names map directly to file names, e.g. `user` -> `icons/user.svg`.
- All icons are outline SVGs from Lucide.
